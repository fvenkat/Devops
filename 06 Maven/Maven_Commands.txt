Initialize a new Maven project
===============================
mvn archetype:generate -DgroupId=io.stockgeeks -DartifactId=project-example

$ mvn clean
$ mvn validate
$ mvn compile
$ mvn test
$ mvn package
$ mvn verify
$ mvn install
$ mvn deploy
$ mvn -Dproject.build.sourceEncoding=UTF-8 -Dproject.reporting.outputEncoding=UTF-8 clean deploy
